# Système de Gestion Hospitalière

## Structure
hospital-management-system/
- README.md
- .gitignore
- src/HMS.c (Code source)
- bin/HMS.exe (Programme exécutable)
- docs/ (Documentation)

## Utilisation
Téléchargez le dossier `bin/` et exécutez `HMS.exe`

## Fonctionnalités
- Gestion des patients et médecins
- Système de files d'attente
- Recherche par arbre binaire
- Sauvegarde des données
